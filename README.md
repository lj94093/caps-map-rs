# caps-map-rs

## Summary
map the CapsLock+i to up etc... inspired by https://github.com/oblitum/caps2esc

details:
- CapsLock+u => home
- CapsLock+i => up
- CapsLock+o => end
- CapsLock+j => left
- CapsLock+k => down
- CapsLock+l => right
- CapsLock+n => backspace
- CapsLock+m =delete

## dependencies
- pkg-config or pkgconf
- libudev
- libevdev
